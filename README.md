# GPODT – Get Pass Or Die Tryin' ☕

Aplikacja do nauki Javy na kolokwium. 299 pytań, 9 kategorii.

## Tryby
- **Fiszki** – przeglądaj pytania z wyjaśnieniami
- **Quiz** – 4 odpowiedzi, wybierz poprawną
- **Próbny egzamin** – 30 losowych pytań, próg 60%

## Kategorie
Klasy abstrakcyjne, Rekordy, Interfejsy, Lambdy, Enum, Generyki, Kolekcje, Stream API, Wielowątkowość

## Uruchomienie
```bash
cd app
npm install
npm run dev
```

Otwórz http://localhost:5173
